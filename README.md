# 💰 Loan Default Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents a **Power BI dashboard** focused on analyzing loan default patterns.

The goal is to identify high-risk customers and understand key factors contributing to loan defaults.

---

## 🛠 Tools Used

* Power BI
* Data Visualization
* Financial Data Analysis

---

## 📊 Dashboard Features

* 📉 Default vs Non-default comparison
* 👤 Borrower demographics analysis
* 💰 Income vs Loan amount insights
* 📊 Risk segmentation

---

## 📈 Key Insights

* Identified high-risk customer segments
* Observed relationship between income and default rate
* Analyzed factors influencing loan repayment behavior

---

## ▶️ How to Use

1. Download `.pbix` file
2. Open in Power BI Desktop
3. Explore dashboard using filters

---

## 🚀 Future Improvements

* Add predictive model for default risk
* Integrate real-time financial data
* Enhance segmentation using ML
